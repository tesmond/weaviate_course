# Weaviate Learning Course

This repository is used to complete the hands-on learning materials and examples for the **Weaviate Academy** course.


## Getting Started

### Prerequisites

- Python 3.11+

## Tools

This project uses modern Python development tools:

- **Hatch** - Project build system and package manager
- **Ruff** - Fast Python linter and formatter
- **isort** - Import statement organizer
- **mypy** - Static type checker
